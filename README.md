# Mini Compiler

## Description-

 This is a Mini Compiler web-based application developed to demonstrate the basic working principles of a compiler.
The system allows users to write simple source code consisting of variable assignments, arithmetic expressions, and print statements. The compiler performs lexical analysis, syntax checking, and execution, and then displays either the output or detailed error messages.

## It includes :

Web-based source code editor

Lexical Analyzer (Tokenization)

Syntax and Runtime Error Detection

Arithmetic Expression Evaluation

PRINT Statement Execution

Output Display Panel
